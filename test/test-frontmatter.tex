\chapter{Introduction}

This document demonstrates that the frontmatter templates are accessible
when the class is installed locally.

The templates are found automatically because they are installed to
\texttt{\textasciitilde/texmf/tex/latex/textured/frontmatter/}

\section{Template Access}

Users can either:
\begin{itemize}
  \item Use \verb|\include{textured/frontmatter/title}| (as in this document)
  \item Copy templates to their project and customize them
  \item Create their own frontmatter using the class commands
\end{itemize}
